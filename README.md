# Financ

Uma aplicação web desenvolvida para ajudar no controle e organização das finanças pessoais de forma simples e eficiente.

## Sobre o projeto

O **Financ** nasceu com uma proposta direta: tornar o acompanhamento financeiro mais claro no dia a dia.

A aplicação permite que o usuário visualize e organize suas movimentações, trazendo mais controle sobre receitas, despesas e decisões financeiras.

Mais do que um exercício técnico, o projeto foi pensado como um produto real, com foco em usabilidade, clareza e experiência do usuário.

## Funcionalidades

- Autenticação de usuários
- Registro e controle de receitas e despesas
- Visualização de informações financeiras
- Interface limpa e intuitiva
- Acesso de teste para exploração da aplicação
